# Project: KU Strategy Presentations

เว็บรวมงานนำเสนอ รายงาน คู่มือ และ dashboard สำหรับงานภายในกองบริหารยุทธศาสตร์ สำนักงานมหาวิทยาลัย มหาวิทยาลัยเกษตรศาสตร์

## เป้าหมาย

- รวมงานนำเสนอและรายงานที่ต้องแชร์เป็นลิงก์
- ใช้ GitHub Pages เป็นพื้นที่เผยแพร่หน้าเว็บ
- แยกข้อมูลอ่อนไหวออกจากพื้นที่สาธารณะ
- เตรียมต่อยอดเป็น dashboard หรือคู่มือระบบภายใน

## ขอบเขตงานระยะแรก

- หน้าแรกของเว็บรวมงานนำเสนอ
- หมวดงาน HR
- หมวดห้องประชุม
- หมวด automation และระบบหลังบ้าน
- หมวดระบบที่เกี่ยวข้อง เช่น KUSmartP, e-Saraban และระบบจองห้องประชุม

## งานที่ควรทำต่อ

1. เพิ่มหน้าแยกสำหรับงาน HR ประจำเดือน
2. เพิ่มหน้าแยกสำหรับการดูแลห้องประชุม
3. เพิ่มหน้าแยกสำหรับ automation/Google Apps Script
4. เพิ่ม template สำหรับรายงานหรือ dashboard
5. ตรวจซ้ำก่อนเผยแพร่ทุกครั้งว่าไม่มีข้อมูลบุคลากรหรือข้อมูลภายในที่อ่อนไหว

## ข้อควรระวัง

repo นี้ใช้ GitHub Pages และอาจเป็น public ได้ จึงไม่ควรใส่ข้อมูลต่อไปนี้ลงไปโดยตรง:

- รายชื่อบุคลากรพร้อมข้อมูลส่วนตัว
- วันลา เวลาเข้าออกงาน เงินเดือน หรือผลประเมิน
- ข้อมูลสุขภาพหรือเอกสารประวัติ
- ไฟล์ภายในที่ยังไม่ได้รับอนุญาตให้เผยแพร่
