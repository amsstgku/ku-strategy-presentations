# KU Strategy Presentations

เว็บรวมงานนำเสนอ รายงาน และคู่มือสำหรับงานภายในกองบริหารยุทธศาสตร์

## ใช้งาน

เปิดไฟล์ `index.html` ใน browser ได้ทันที หรือเปิดผ่าน GitHub Pages

## GitHub Pages

1. ไปที่ Settings > Pages
2. เลือก Source เป็น `Deploy from a branch`
3. เลือก branch `main` และ folder `/root`
4. กด Save

หลังจากระบบ deploy แล้ว เว็บจะอยู่ที่:

`https://amsstgku.github.io/ku-strategy-presentations/`

## หมายเหตุเรื่องข้อมูล

ก่อนเผยแพร่ไฟล์จริง ควรแยกข้อมูลอ่อนไหว เช่น รายชื่อบุคลากร วันลา เงินเดือน ผลประเมิน ข้อมูลสุขภาพ และแฟ้มประวัติ ออกจากพื้นที่สาธารณะ
